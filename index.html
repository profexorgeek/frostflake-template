This starter project demonstrates many of the features of the FrostFlake game engine. It rotates through demo views every few seconds. Here's a brief description of each view:

  1. AnimationDemo - demonstrates frame-based animation from a spritesheet.
  2. AudioDemo - demonstrates loading and playing sounds. Click to play a sound.*
  3. CollisionDemo - demonstrates collisions with engine physics.
  4. InputDemo - demonstrates input handling. Press WASD to move the camera.
  5. ManySpritesDemo - demonstrates sprite loading and load-tests rendering.
  6. ParentChildDemo - demonstrates parent-child relationships in the scene graph.
  7. RenderTargetDemo - demonstrates rendering a custom sprite texture.
  8. ShapesDemo - demonstrates rendering pure shape objects and load-tests rendering.
  9. TextStringDemo - demonstrates rendering string of text.

* Browser restrictions require user input before audio can play. If the mouse has not moved since the game loaded, audio will throw an exception.
